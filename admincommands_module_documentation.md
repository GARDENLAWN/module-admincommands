# GardenLawn AdminCommands Module Documentation

This document provides an overview of the GardenLawn AdminCommands module.

## Directory Structure

- **Block**: Block classes.
- **Console**: Console commands.
- **Controller**: Controller actions.
- **etc**: Configuration files.
- **view**: Layouts and templates.

## Overview

This module adds custom admin commands and functionalities to the Magento backend, streamlining administrative tasks.
